# useTitle

改变浏览器 title 的 Hook

<preview path="./demo/index.vue" title="基本使用" description='改变浏览器 title,可动态改变； '></preview>

## API

## 参数

| 参数  | 说明     | 类型          | 默认值 |
| ----- | -------- | ------------- | ------ |
| title | 页面标题 | `Ref<string>` | -      |

## Options

| 参数             | 说明                               | 类型      | 默认值  |
| ---------------- | ---------------------------------- | --------- | ------- |
| restoreOnUnmount | 组件卸载时，是否恢复上一个页面标题 | `boolean` | `false` |
