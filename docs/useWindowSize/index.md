# useWindowSize

获取浏览器窗口的视口宽度和高度，并在窗口大小变化时自动更新。

<preview path="./demo/index.vue" title="基本使用" description='useWindowSize'></preview>

## API

### 返回值

| 参数   | 说明           | 类型           |
| ------ | -------------- | -------------- |
| width  | 浏览器窗口宽度 | _Ref\<number>_ |
| height | 浏览器窗口高度 | _Ref\<number>_ |
