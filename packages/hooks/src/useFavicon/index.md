# useFavicon

设置页面的 favicon。

<preview path="./demo/index.vue" title="基本使用" description='useFavicon'></preview>

## API

```typescript
useFavicon(href: string);
```

## Params

| 参数 | 说明                                                  | 类型          | 默认值 |
| ---- | ----------------------------------------------------- | ------------- | ------ |
| href | favicon 地址, 支持 `svg`/`png`/`ico`/`gif` 后缀的图片 | `Ref<string>` | -      |
